## 🚀 1. Rodar o Frontend

### Pré-requisitos:
- Node.js 18+
- Yarn ou NPM

```bash
cd frontend
yarn install
yarn dev
```

Rodar a aplicação usando
```bash
npm run dev
```

A aplicação será iniciada em: http://localhost:3000
